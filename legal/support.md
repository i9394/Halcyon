## Frequently Asked Questions / 常见问题

**How do I save a video from a web page?**
Long-press the video element. Viohold will offer a "Save to Local" option for supported formats (HLS / DASH / M3U8 / MP4 / TS / WebM / MKV).

**怎么保存网页上的视频?**
长按视频元素即可。Viohold 会为支持的格式 (HLS / DASH / M3U8 / MP4 / TS / WebM / MKV) 弹出「保存到本地」选项。

**Why are some downloads slower than expected?**
Free tier supports 1 concurrent task with 1 thread. Pro unlocks up to 6 concurrent tasks and 12 threads per task.

**为什么有时下载速度不如预期?**
免费版支持 1 个并发任务、单任务 1 线程。Pro 解锁最多 6 个并发任务、单任务最多 12 线程。

**How do I restore my Pro purchase on a new device?**
Open the app → Settings → Viohold Pro → **Restore Purchases**. Sign in with the same Apple ID used for the original purchase.

**新设备上如何恢复 Pro 购买?**
打开 App → 设置 → Viohold Pro → **恢复购买**。使用购买时同一 Apple ID 登录即可。

**How do I cancel a subscription?**
System Settings → Apple ID → Subscriptions → Viohold Pro → Cancel.

**如何取消订阅?**
系统设置 → Apple ID → 订阅 → Viohold Pro → 取消订阅。
